First you need to download the corresponding years data from https://www.worldbank.org/en/programs/lsms/initiatives/lsms-ISA#8 and unzip the files and follow the same folder structures alluded to in the cleaning scripts `2010_2011_data.R`, `2011_2012_data.R`, `2018_2019_data.R`. Run the cleaning scripts, then you can use the analysis scripts -- `panel_analysis_script.R` and `full_analysis_script.R`.
